### Linux

#### Ubuntu

Run the following to install dependencies:

```
sudo apt-get update -y && sudo apt-get upgrade -y
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

#### Fedora

Run the following to install dependencies:

```
sudo dnf upgrade -y
sudo dnf install curl git unzip xz zip mesa-libGLU
```
